在Windows上安装Hugo

本篇文章的目标是详细记录在 我的Windows 电脑上安装 Hugo 的过程

设置所使用的文件夹

需要一个存储 Hugo 可执行文件、博客内容（你创建的的那些文件），以及生成文件（Hugo 为你创建的 HTML）的地方。

  1. 创建一个新的文件夹，D:\Hugo。
  2. 创建一个新的文件夹，D:\Hugo\bin。
  3. 创建一个新的文件夹，D:\Hugo\Sites。

下载预先编译好的 Windows 版本的 Hugo 可执行文件

  1. 在浏览器中打开 https://github.com/spf13/hugo/releases。
  2. 我下载的版本是hugo_extended_0.110.0_windows-amd64.zip
  3. 并保存到 D:\Hugo\bin 文件夹中，并从中提取所有的文件，可以看到有一个 hugo.exe。
  4. 确保 hugo.exe 文件在 D:\Hugo\bin 文件夹。
  5. 把 hugo.exe 可执行文件添加到你的 PATH路径中。

高级系统设置»环境变量»Path»编辑»新建，然后添加 D:\Hugo\bin

验证可执行文件

  1. 打开一个命令提示符窗口。
  2. 在提示符中，输入 hugo help 并按下 Enter 键。你应该能够看到输出

生成网站

  1. 在命令提示符中，跳转当前目录到 Sites 文件夹。
  2. 输入hugo new site flower.com并按下 Enter 键,生成名为flower.com的网站

新生成的网站结构如下：

  • config.toml 是配置文件，在里面可以定义博客地址，构建配置，标题，导航栏等等。
  • themes是主题目录，可以下载喜欢的主题，然后配置在config.toml里面。
  • content是博客文章的目录。
  • static是博客的静态资源，比如图片
  • public是编译后的目标文件的目录

3.在最终博客可以运行之前，需要先配置一个主题。

官方主题网站：https://themes.gohugo.io/
